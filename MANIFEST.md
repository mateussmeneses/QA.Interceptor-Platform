# QA.Interceptor Platform Manifest

QA.Interceptor Platform exists to empower QA teams with practical traffic interception and API validation tools.

## Principles

- QA-first product decisions
- Free and open ecosystem
- No vendor lock-in
- Local-first and privacy-first execution
- Simplicity over unnecessary complexity

## Feature Filter

Every proposed feature must answer:

"Does this improve real QA validation workflows?"

If not, it does not belong in the product scope.
